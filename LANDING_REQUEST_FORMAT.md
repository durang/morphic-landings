# LANDING REQUEST FORMAT (v1)

Formato recomendado para pedir una landing:

OBJETIVO=leads | ventas | registro | demo
NICHO=<nicho exacto>
OFERTA=<qué se ofrece + precio si aplica>
CTA_TEXT=<texto del botón>
CTA_LINK=<url final>
ESTILO=BRAND_MORPHIC (default) | custom
IDIOMA=es | en
SLUG=<slug deseado>

Notas:
- Si falta SLUG, se propone uno.
- Si falta CTA_LINK, se pregunta.
- Sin MORPHIC_APPROVE_BUILD no se crea nada en repo.
