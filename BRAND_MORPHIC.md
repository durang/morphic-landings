# MORPHIC LABS — BRAND DNA (v1)

## Identidad
Marca premium de automatización, IA y sistemas digitales.
Nada genérico. Nada barato. Nada de “curso”.

## Sensación
Poder, claridad, control, velocidad.
El usuario siente que entra a un sistema serio, no a marketing.

## Estilo Visual
- Dark mode real (no gris claro)
- Acentos neón sutiles
- Espacios amplios
- Cards con borde fino y blur ligero

## Colores
Fondo: #07070A / #0B0B0F
Texto: #E9E9F1
Neón primario: #00FF9D
Neón secundario: #0066FF
Bordes: rgba(255,255,255,.08)

## Tipografía
System UI o Inter.
H1 grandes, agresivos.
Párrafos cortos.

## Copy
Directo, sin relleno.
Autoridad tranquila.
Frases cortas.
Nada de emojis.
Nada de promesas vagas.

## Estructura Base de Landing
1. Hero (promesa clara + CTA)
2. Problema real (dolor específico)
3. Sistema / Solución (cómo funciona)
4. Proof (números, casos, lógica)
5. Oferta clara
6. FAQ
7. CTA final

## Prohibido
- Plantillas genéricas
- Copy tipo “coach”
- Demasiado texto
- CTAs confusos
